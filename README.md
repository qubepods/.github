# `.github`

Organization-level GitHub defaults for [**qubepods**](https://qubepods.com) —
the hosted runner for [`q64`](https://q64.dev) WebAssembly qubes.

## What's here

| Path | Purpose |
|---|---|
| [`profile/README.md`](./profile/README.md) | The org profile shown at [github.com/qubepods](https://github.com/qubepods). |

This repo holds shared, org-wide GitHub configuration (the public profile, and
any future issue/PR templates or community health files that apply across every
qubepods repository). Per-project code lives in the individual repos.
